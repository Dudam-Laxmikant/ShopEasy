# React + Vite + Tailwind CSS

This is a modern frontend project built with React, Vite, and Tailwind CSS.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **React 18.3** - A JavaScript library for building user interfaces
- **Vite 6.0** - Next generation frontend tooling
- **Tailwind CSS 3.4** - A utility-first CSS framework
- **ESLint** - Code linting and quality checks
